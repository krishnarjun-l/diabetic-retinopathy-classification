# Diabetic Retinopathy Classification using Deep Learning

## 📌 Project Overview
Developed an automated system to classify the severity of Diabetic Retinopathy (DR) from fundus images. This project aims to assist medical professionals in early detection, potentially preventing vision loss in diabetic patients.

## 📊 Dataset
* **Source:** Kaggle Diabetic Retinopathy Detection 
* **Size:** 35,108 high-resolution fundus images 
* **Pre-processing:** Image resizing, normalization, and data augmentation to handle class imbalance.

## 🛠️ Tech Stack
* **Language:** Python 
* **Frameworks:** TensorFlow, Keras 
* **Libraries:** NumPy, Pandas, Matplotlib, OpenCV 
* **Model Architectures:** EfficientNetB0, MobileNetV2, InceptionV3 

## 🚀 Key Results
* **Top Performing Model:** EfficientNetB0 
* **Accuracy:** achieved **76%** on the test set 
* **Insight:** Conducted comparative analysis between various CNN architectures to identify the most robust model for medical imaging.
